# WhiruS Portfolio

A modern and fully responsive developer portfolio built with React and Tailwind CSS to showcase my projects, skills, journey, and contact information. Designed with reusable components, interactive UI, custom branding, and smooth user-focused experiences.

---

## 🚀 Features

- Fully responsive design across all screen sizes
- Modern UI with smooth interactions
- Interactive animated hero section
- Project showcase section
- Skills section
- About / journey section
- Contact section
- Custom branding and project logos
- Reusable component-based architecture
- Optimized for performance

---

## 🛠 Tech Stack

### Frontend
- React.js
- JavaScript (ES6+)
- Tailwind CSS
- React Router DOM
- React Icons

### Build Tool
- Vite

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/WhiruS-X07/whirus-portfolio.git
```

Go to project folder:

```bash
cd whirus-portfolio
```

Install dependencies:

```bash
npm install
```

Run development server:

```bash
npm run dev
```
